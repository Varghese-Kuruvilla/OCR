# Optical Character Recognition for Financial Documents
This repository gives an example of OCR applied to a bank loan application.

#### Requirements

------

pytesseract==0.3.2 
numpy==1.14.5
opencv_contrib_python==4.0.0.21
pandas==0.23.4
matplotlib==3.0.0
Pillow==7.0.0

------

#### Usage 

python tesseract.py
